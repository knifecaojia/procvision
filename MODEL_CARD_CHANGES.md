# ModelCard 组件设计更新

## 概述
根据模型管理页面设计规范，已将 ModelCard 组件重构为独立的可复用组件，并应用了深色主题的工业风格设计。

## 文件结构

```
src/
├── ui/
│   ├── components/
│   │   ├── __init__.py              (更新)
│   │   └── model_card.py            (新建 - 独立组件)
│   ├── pages/
│   │   └── model_page.py            (更新 - 导入独立组件)
│   └── main_window.py               (更新 - 添加样式)
└── test_model_card.py               (新建 - 测试脚本)
```

## 主要更改

### 1. 创建独立的 ModelCard 组件
**文件**: `src/ui/components/model_card.py`

#### 特性
- 完全独立的可复用组件
- 支持动态模型类型（OpenCV / YOLO）
- 响应式布局和深色主题
- 悬停效果和状态管理

#### 组件结构
```python
class ModelCard(QFrame):
    - 头部区域（图标、标题、版本、状态标签）
    - 描述区域
    - 信息网格（类型、大小、更新时间）
    - 操作按钮组（查看、更新、删除）
```

### 2. 重构 model_page.py
**文件**: `src/ui/pages/model_page.py`

#### 更改
- 移除内嵌的 `ModelCard` 类定义
- 添加导入: `from ..components.model_card import ModelCard`
- 更新图标为表情符号:
  - OpenCV: `🖥️` (蓝色主题)
  - YOLO: `🧠` (紫色主题)
- 保持所有示例数据和功能

### 3. 更新组件导出
**文件**: `src/ui/components/__init__.py`

```python
from .model_card import ModelCard

__all__ = ['ModelCard']
```

### 4. 添加样式系统
**文件**: `src/ui/main_window.py`

在 `setup_style()` 方法中添加了完整的 ModelCard 样式系统:

#### 颜色系统
- **卡片背景**: `#252525` (深灰色)
- **边框**: `#3a3a3a` (中灰色)
- **悬停边框**: `rgba(255, 165, 0, 0.5)` (半透明橙色)
- **信息区域**: `#1a1a1a` (更深灰色)

#### 模型类型主题
- **OpenCV**: 蓝色系
  - 背景: `rgba(59, 130, 246, 0.1)`
  - 图标: `#60a5fa` (蓝色)
- **YOLO**: 紫色系
  - 背景: `rgba(168, 85, 247, 0.1)`
  - 图标: `#c084fc` (紫色)

#### 状态标签
- **启用**: 绿色主题
  - 背景: `rgba(34, 197, 94, 0.1)`
  - 文字: `#4ade80` (绿色)
- **未用**: 灰色主题
  - 背景: `rgba(107, 114, 128, 0.1)`
  - 文字: `#9ca3af` (灰色)

#### 按钮样式
- **查看按钮**: 透明背景 + 边框，悬停时显示背景
- **更新按钮**: 橙色背景 (#f97316)，悬停时加深 (#ea580c)
- **删除按钮**: 红色边框，悬停时显示红色背景
