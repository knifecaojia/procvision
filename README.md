# SMART-VISION 工业视觉系统

基于 PySide6 的工业视觉桌面应用，提供登录认证、相机管理与工艺执行等能力。

## 使用指南

### 创建虚拟环境（venv）

- Windows：
  - `python -m venv venv`
  - `venv\Scripts\activate`
  - `pip install -r requirements.txt`

- Linux/macOS：
  - `python -m venv venv`
  - `source venv/bin/activate`
  - `pip install -r requirements.txt`

### 启动完整应用

- `python run_app.py`
